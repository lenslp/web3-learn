优化前：
第一次构建：531s
第二次构建：434s
第三次构建：391s
第四次构建：488s
优化后：

### 优化方法
1. thread-loader开启多线程处理（webpack4环境下反而更慢，webpack5环境效果好一些）
2. 设置forkTSChecker：false，关闭ts类型检查（没啥效果）
3. nodeModulesTransform：关闭node_modules处理，提升构建速度。注：部分依赖不做处理会报错（没啥效果）
4. 配置 externals，对于一些大尺寸依赖，比如图表库、antd 等，可尝试通过 externals 的配置引入相关 umd 文件，减少编译消耗，比如 react 和 react-dom。
5. 将terser压缩器替换为esbuild，提升压缩速度。
6. ci缓存：缓存node_modules和构建缓存，避免重复构建。
```typescript
cache:
  paths:
    - node_modules/
    - node_modules/.cache/
```
