## 三列布局
1. grid布局
   ```css
   .container {
     display: grid;
     grid-template-columns: 1fr 1fr 1fr; // 或者：repeat(3, 1fr)
     gap: 20px; // 列间距
   }
   ```
2. flex布局
   ```css
   .container {
     display: flex;
   }
   .item {
     flex: 1;
   }
   ```
3. 浮动布局
   ```css
   .container {
     overflow: hidden;
   }
   .item {
     float: left;
     width: 33.33%;
   }
   ```
