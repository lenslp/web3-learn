## 项目背景：
为运营团队提供用户行为、交易数据的可视化分析工具，支持实时监控、自定义报表、指标预警，辅助业务决策，覆盖 DAU / 交易量 / 充值提现等核心指标。

## 技术栈：
React 18 + TypeScript + Next.js + Recharts + Ant Design + SWR + Tailwind CSS

## 主要职责与实现：
+ 实时数据看板与可视化
开发「全局运营看板」：用 Recharts 实现 DAU/MAU 趋势线图、成交量饼图、用户地域分布热力图，数据每 30 秒自动刷新（SWR 配置 revalidateInterval），异常波动（如交易量突增 200%）用红色警戒线标注。
+ 封装「指标卡片组件」：展示核心数据（如总用户数、24h 交易额），支持环比 / 同比计算（如 "较昨日 + 12%"），数值变化时添加数字滚动动画（用 Framer Motion 实现），提升数据感知力。
+ 自定义报表
实现「报表生成器」：用户通过拖拽选择维度（时间 / 地区 / 用户等级）和指标（交易量 / 充值金额），动态生成表格 / 图表，支持导出 Excel（调用 SheetJS 库），复杂报表生成时间从 3 分钟优化至 15 秒。
+ 性能与体验优化
采用 Next.js SSR 渲染首屏看板，提升加载速度；非首屏组件用动态导入（next/dynamic）实现懒加载，减少初始 JS 体积。
+ 实现「离线缓存」：用 IndexedDB 缓存历史报表数据，网络中断时自动切换至本地数据，确保核心功能可用。
## 工作成果：
+ 提升运营团队数据分析效率，提升用户转化率；
+ 自定义报表工具被运营人员高频使用。


## 为什么前端导出excel
+ 减少服务器压力
+ 即时响应，提升用户体验
+ 支持前端动态数据：用户在报表生成器中调整参数后，立即在前端生成并导出 Excel，无需等待服务器处理。
+ 避免网络波动导致的下载中断、超时，稳定性更高
