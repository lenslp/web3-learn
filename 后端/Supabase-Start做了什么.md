# Supabase Start 做了什么

> 创建日期：2026-02-22  
> 分类：数据库 / Supabase / 工具

## 一句话理解

**`supabase start` = 在本地启动一套完整的 Supabase 开发环境**，包括数据库、认证、存储、边缘函数等所有服务。

## `supabase start` 做了什么？

### 执行流程

```bash
$ supabase start
```

```
┌─────────────────────────────────────────────┐
│           supabase start 执行步骤                  │
└─────────────────────────────────────────────┘

Step 1: 检查 Docker
┌─────────────────────────────────────────┐
│  确认 Docker Desktop 是否运行             │
│  确认 Docker Compose 是否安装            │
└─────────────────────────────────────────┘
              │
              ▼
Step 2: 读取配置
┌─────────────────────────────────────────┐
│  读取 supabase/config.toml               │
│  读取 .env 文件                          │
│  获取配置参数                            │
└─────────────────────────────────────────┘
              │
              ▼
Step 3: 启动 Docker 容器
┌─────────────────────────────────────────┐
│  docker-compose up -d                    │
│  启动所有 Supabase 服务                  │
└─────────────────────────────────────────┘
              │
              ▼
Step 4: 初始化数据库
┌─────────────────────────────────────────┐
│  创建 PostgreSQL 数据库                   │
│  执行迁移脚本                            │
│  加载初始数据                           │
└─────────────────────────────────────────┘
              │
              ▼
Step 5: 输出信息
┌─────────────────────────────────────────┐
│  显示服务地址                           │
│  显示 API 密钥                          │
│  显示本地管理后台地址                   │
└─────────────────────────────────────────┘
```

## 本地开发流程

```
┌─────────────────────────────────────────────────────────────┐
│              Supabase 本地开发工作流                           │
└─────────────────────────────────────────────────────────────┘

1. 启动本地环境
   $ supabase start

2. 本地开发
   ┌─────────────┐
   │  写代码    │
   │  连接 localhost:8000  │
   │  使用 anon key     │
   └─────────────┘

3. 测试
   ┌─────────────┐
   │  本地测试   │
   │  localhost:54323  │
   │  打开 Studio 管理  │
   └─────────────┘

4. 部署到云端
   $ supabase db push     # 推送数据库变更
   $ supabase functions deploy  # 部署边缘函数
   $ supabase link --project-ref xxx  # 关联云端项目
```

## 常用命令

```bash
# 启动本地环境
supabase start

# 停止本地环境
supabase stop

# 重置本地数据库
supabase db reset

# 推送数据库变更
supabase db push

# 创建迁移
supabase migration new my_migration

# 查看本地状态
supabase status

# 打开本地管理后台
# 浏览器访问 http://localhost:54323
```
